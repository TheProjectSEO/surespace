The attribution gap is the whole game. Your website pulls 210 new visitors from Organic Search every month. Your app picks up 214-224 installs. Today nothing connects them — the App Store / Play Store buttons aren't tagged, and stores don't pass attribution into Firebase by default. So when you ask "what's SEO actually delivering?", the honest answer today is "we can't count it." Phase 1 builds the deferred-deep-link infrastructure (smart-link attribution + UTM tagging on store buttons) that makes web→install countable for the first time. Specific tool selection (Branch / AppsFlyer / Adjust / Firebase-native) is a Week-1 onboarding decision with you and Naman based on cost, privacy posture, and stack fit. Every Gate KPI below assumes this layer is built — vendor-neutral.

Three phases. No silent months.

Every month has a visible deliverable and a measurable output. The first 30 days set up the foundation we need to be honest about what's working and what isn't.

  • 25+ indexable pages live, all 7 schema types deployed
